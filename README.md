🦕 Dino Run Remastered
A faithful recreation of the Chrome Dino game — fully browser-based, zero dependencies.
Play Online
If hosted on : https://editerpros.github.io/dino.run.remastered/
Features

🦕 Pixel-art dino with run, jump & duck animations
🌵 3 cactus obstacle types (single, cluster, wide)
🦅 Pterodactyls unlock after score 250
🌙 Day → Night cycle with stars
🎵 Retro sound effects
📱 Touch/tap support for mobile
🏆 High score tracking

Controls
> Start/Restart--Space, ↑, or Tap
> Jump--Space or ↑
> Duck--↓
